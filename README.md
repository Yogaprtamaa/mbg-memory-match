# MBG Memory Match

Memory Matching Game bertema Program Makan Bergizi Gratis (MBG).

## Teknologi
- Python 3.13
- Pygame 2.6.1

## Struktur Project

```
assets/
src/
  cards/
  config/
  managers/
  objects/
  scenes/
  utils/
```

## Menjalankan Project

Aktifkan virtual environment:

```powershell
.venv\Scripts\Activate.ps1
```

Install dependency:

```powershell
pip install -r requirements.txt
```

Jalankan program:

```powershell
python src/main.py
```
